# Installation FACTORIE is known to run on MacOS, Linux, and Windows with Cygwin, and is expected to run on any platform on which the Java Virtual Machine is available. ## Coding Standards We generally follow the Scala style guide suggestions in http://davetron5000.github.com/scala-style/index.html For indentation use two spaces rather than tabs.
