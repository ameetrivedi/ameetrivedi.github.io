FACTORIE

is a toolkit for deployable probabilistic modeling, implemented as a software library in Scala. It provides its users with a succinct language for creating relational factor graphs, estimating parameters and performing inference.

Learn more »

Inference

The basic use of a graphical model is to perform inference: making predictions about the values of unobserved variables, conditioned on the values of observed variables and the parameters. FACTORIE has implementations of many common inference algorithms.

Learn more »

Learning

Much of modern machine learning finds parameters by setting up an optimization problem trading off having low error on the training data with having a simple predictor which should generalize well. FACTORIE includes an optimization package directly designed to solve optimization problems of this kind.

Learn more »

Natural language processing

FACTORIE has been successfully applied to various tasks in natural language processing and information integration, including named entity recognition, entity resolution, relation extraction, parsing, and latent-variable generative models, including latent Dirichlet allocation.
